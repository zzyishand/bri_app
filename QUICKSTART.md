# 快速开始指南 🚀

## 5分钟快速上手

### 步骤1: 安装依赖 (1分钟)

```bash
cd bri_app
pip install -r requirements.txt
```

### 步骤2: 启动应用 (30秒)

**Windows用户:**
```bash
run.bat
```

**Linux/Mac用户:**
```bash
chmod +x run.sh
./run.sh
```

浏览器会自动打开 `http://localhost:8501`

### 步骤3: 首次更新数据 (3-5分钟)

1. 点击左侧 **"🔄 Data Update"**
2. 点击 **"🔍 Check for Updates"** 按钮
3. 点击 **"▶️ Start Update"** 按钮
4. 等待进度条完成（约3-5分钟）

### 步骤4: 查看Dashboard (30秒)

1. 点击左侧 **"📊 Dashboard"**
2. 查看所有资产的泡沫风险热力图
3. 点击任何资产查看详细分析

## 完成！🎉

现在您可以：
- ✅ 实时监控26个资产的泡沫风险
- ✅ 每天快速更新最新数据（增量更新，只需几秒）
- ✅ 查看历史趋势和详细指标

## 日常使用

每天只需3步：
1. 打开应用 → `run.bat` 或 `run.sh`
2. 进入 "🔄 Data Update" → 点击更新
3. 回到 "📊 Dashboard" → 查看最新BRI
